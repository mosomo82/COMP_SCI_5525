A terminal-based package tracking CLI built with Python stdlib only, containerized with Docker and published to Docker Hub under the student's own account (mosomo82). No third-party pip packages are used. The application runs identically locally and inside a python:3-alpine Docker container.

Steps & Screenshots

1

Build the Docker Image — image name contains student name

The image name tonyndemoimage embeds the student name (Tony N).
